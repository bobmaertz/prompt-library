# Shared Resources

This directory contains prompts, templates, and resources that work across both Claude Code and Cursor.

## Structure

- `prompts/` - Universal prompts that can be referenced by both tools
- `templates/` - Code templates and scaffolding
- `docs/` - Shared documentation and guidelines

## Usage

These resources can be referenced from both Claude and Cursor configurations, reducing duplication while maintaining tool-specific customizations.
